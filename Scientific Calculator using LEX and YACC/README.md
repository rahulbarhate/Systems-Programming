# Scientific Calculator using LEX and YACC

Install bison and flex using: -

> sudo apt-get install bison

> sudo apt-get install flex

## Instructions

### To Run:

> lex c1.l

> yacc c1.y

> gcc y.tab.c -ly -lfl -lm

> ./a.out
